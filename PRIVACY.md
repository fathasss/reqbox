# Privacy Policy – ReqBox

Last updated: March 2026

ReqBox respects your privacy. This Privacy Policy explains how the ReqBox application handles user data.

## Information Collection

ReqBox does **not collect, store, or share any personal information** from users.

The application is designed as a developer tool for testing HTTP requests and APIs. Any data entered into the application (such as URLs, headers, request bodies, or API responses) is processed locally on the user’s device.

## API Requests

ReqBox allows users to send HTTP requests to external APIs. These requests are made directly from the user's device to the target server specified by the user.

ReqBox does **not intercept, store, or transmit this data to any third-party servers controlled by the developer**.

## Data Storage

If the application provides features such as saving request history or configurations, this data is stored **locally on the user's device** and is not transmitted to external servers.

## Third-Party Services

ReqBox does not use third-party analytics tools, advertising networks, or tracking technologies.

## Security

While ReqBox does not collect user data, users should be cautious when sending sensitive information through API requests, as the security of the target API server is outside the control of this application.

## Children's Privacy

ReqBox is not intended for use by children under the age of 13. The application does not knowingly collect personal information from children.

## Changes to This Privacy Policy

This Privacy Policy may be updated from time to time. Any updates will be reflected by changing the "Last updated" date.
